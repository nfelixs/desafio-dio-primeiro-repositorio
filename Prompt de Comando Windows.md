# Prompt de Comando - Windows 

Primeiro contato com Prompt de Comando no meu Notebook :computer:

#### Abrindo Prompt :open_file_folder:

- Clicar no "Iniciar" do seu computador
- Escrever "cmd"
- Abrir Prompt de Comando

#### Comandos básicos 

- dir: Listar/situar dentro do sistema operacional entendendo onde estou

- cd + nome_da_pasta: Navega para dentro desta pasta

- cd .. : Retrocede navegação (_Obs.: Utilizar espaço antes dos dois pontos_)

- cls: Limpar tela

- Tecla Tab: Função de autocompletar (_Exemplo: W + Tab = Windows_)

- mkdir + nome_da_pasta: Criar pasta/diretório

- echo > nome.txt: Criar arquivo (_Obs.: Vale para .txt; .md; etc_)

- del + noma_da_pasta: Deletar arquivos

- rmdir + nome_da_pasta + /S /Q: Deletar repositório/diretório e tudo incluso nele

- Seta pra cima: Navega em todos os comandos que já digitei naquele momento
